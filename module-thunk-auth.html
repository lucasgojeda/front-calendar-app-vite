Module: thunk-auth

En este archivo encontramos todas las acciones relacionadas a la autenticación.

Methods

async static startChecking()

Esta función realiza el chequeo de si un usuario está logueado mediante la autenticación por token cada vez que recarga la pagina.

async static StartLogin(correo, password)

Esta función realiza el inicio de sesión.

Parameters:
Name Type Description
correo String Correo del usuario.
password String Contraseña del usuario.

async static startLogout()

Esta función realiza el cierre de sesión en la aplicación limpiando el localStorage y los datos del usuario del store en redux.

async static startRegister(name, correo, password, role)

Esta función realiza el registro de un nuevo usuario.

Parameters:
Name Type Description
name String Nombre del nuevo usuario.
correo String Correo del nuevo usuario.
password String Contraseña del nuevo usuario.
role String Role del nuevo usuario (usuario por defecto).
