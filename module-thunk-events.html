Module: thunk-events

En este archivo encontramos todas las acciones relacionadas con los eventos.

Example

evento = {
    title: 'Cumpleaños del jefe',
    start: Date,
    end: Date,
    notes: 'Comprar el pastel',
    user: {
        _id: '123',
        name: 'Fernando'
    }
}
